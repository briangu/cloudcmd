Overview

Photon is a photo sharing app built upon the Social Content Platform (SCP), viper.io and the search-as-a-service platform SIN.

The application is easy to build as it uses vertical feeds queried from SCP to construct public and personal (my) image feeds. Commenting and soon-to-be-completed Liking are inherited behaviors from SCP.
